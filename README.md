# pi calculation with ancient VM

### Run code

- `git clone https://github.com/0xF6/pi.git`  
- `cd pi` 
- `yarn global add @rune-temp/cli`    
- `rune restore`   
- `rune vm install`   
- `rune vm install compiler`  
- `rune vm`
